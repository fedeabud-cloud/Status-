# KÖNIG | Seguimiento de Tareas

Proyecto **KÖNIG | Seguimiento de Tareas** — aplicación web estilo Asana para gestionar tareas, prioridades y generar reportes con exportación a Excel y PDF.

## Características principales
- Interfaz en español, tema claro con acentos turquesa y negro grafito.
- Pestañas: **Tareas** y **Reportes**.
- Campos por tarea: Título, Prioridad, Estado, Responsable, Fecha de vencimiento, Progreso.
- Persistencia local en `localStorage` (se conserva al recargar).
- Exportación a Excel (.xlsx) y a archivo de texto (.txt) como versión simple de PDF.
- Listo para subir a GitHub y desplegar en Vercel o StackBlitz.

## Cómo usar
1. Cloná o subí los archivos a un repositorio en GitHub llamado `seguimiento-de-tareas`.
2. En StackBlitz abrí: `https://stackblitz.com/github/tu_usuario/seguimiento-de-tareas` (reemplazá tu usuario).
3. Para ejecutar localmente:
   - `npm install`
   - `npm start`

## Deploy en Vercel
1. Importá el repo en Vercel y desplegá (Vite + React).
